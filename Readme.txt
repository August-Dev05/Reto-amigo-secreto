CHALLENGE Amigo Secreto

Este proyecto es una aplicación web sencilla que permite a los usuarios agregar nombres a una lista y luego sortear 
aleatoriamente un "amigo secreto" entre ellos. Funciona mediante un formulario donde los usuarios ingresan nombres, 
los cuales se almacenan en un array y se muestran dinámicamente en una lista. Al presionar el botón de sorteo, el
programa elige un nombre al azar y lo muestra en una alerta. La interfaz está diseñada con HTML y CSS, mientras que 
la lógica se maneja con JavaScript, asegurando una experiencia interactiva y fácil de usar para organizar sorteos 
entre amigos.

Por: Kevin Escobar
